2D Platformer

This is a 2D Platformer made in Unity game engine. The code is written in C# and 95% of the assets are hand drawn

Implemented Functions

  • Ability to attack and damage enemies
  • Losing Health when damaged by enemies
  • Animations of death, attack, damage, and many more
  • Pathfinding for enemies
  • Patrol for enemies (the enemy characters walk around the map)
  • Signs which display a dialog box when interacted with
  • Shop which contains multiple chests that can be opened by spending coins.
  • Moving Platforms
  • Parallax background which changes according to the speed and direction of the player
  • Inventory (not completely implemeted)
