Controlled Chaos

This was a group project lasting 3 Sprints over the course of the Fall semester

Tasks (User Stories) Implemented by me personally:

  • Ability to attack and damage enemies
  • Healthbars above enemies
  • Implemented explosive weapons such as bombs which cause damage in a wide circle
  • Implemented the inventory and the ability descriptions of items
  • Implemented 2 enemy types the wizard which shoots ranged projectiles and the skeleton which does close range damage
  • A* Pathfinding for all enemies. Tweaked for each specific enemy.
  • Implemented the shop which contains multiple chest that can be opened by spending coins.
  • Each chest retrieves a random item for a specified pool of items
  • A pause menu that can be used to send feedback to the developers
  • An option to choose between different character before entering the game
